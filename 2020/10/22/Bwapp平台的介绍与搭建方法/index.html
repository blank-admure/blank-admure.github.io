安装方法: 在这里，我教大家的是用centos虚拟机和docker下载安装bwapp

有docker的可以跳过此步，安装此环境需要先安装docker

注：以下操作均是以root权限运行的

1.Docker 要求 CentOS 系统的内核版本高于 3.10 ，查看本页面的前提条件来验证你的CentOS 版本是否支持 Docker 。通过 uname -r 命令查看你当前的内核版本

uname -r

2.、初步安装和启动docker

yum update -y
yum -y install docker
systemctl start docker

3.因为docker自带的镜像源不是国内的，比较慢，下载仓库容易出错，所以我们先把它换成国内的镜像源，在这以阿里源为例:

vi /etc/docker/daemon.json
打开文件，填入以下内容：
{
“registry-mirrors”: [“https://aj2rgad5.mirror.aliyuncs.com"]
}

填入完毕后，按esc然后按冒号”:”,在输入x，回车即保存成功。

4.重启docker

systemctl daemon-reload
systemctl restart docker.service

5、测试docker是否正常安装和运行

docker run hello-world

运行成功即为安装成功。
