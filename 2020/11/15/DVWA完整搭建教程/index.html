DVWA完整搭建教程

一. 简介

DVWA（Damn Vulnerable Web Application）是一个很容易受到攻击的PHP / MySQL Web应用程序，其主要目标是帮助安全专业人员在法律环境中测试他们的技能和工具，帮助Web开发人员更好地了解保护Web应用程序的过程。

二. 搭建前准备

  1. phpstudy
    下载官网：https://www.xp.cn/
    下载完成后打开Apache+MySQL功能
    点击网站，再单击左上角的创建网站
    根目录改到DVWA下，点击确定，网站即可创建成功。

    2.DVWA
    下载官网：http://www.dvwa.co.uk/

三. 搭建环境

将下载的DVWA文件解压到PHP study的WWW目录下
解压后进入DVWA下的config文件打开config.inc.php文件

将p@ssw0rd 修改为 root

修改完成后，回到phpstudy的网站界面，单击管理操作，打开网站，单机创建或重置数据库
着会跳转到DVWA的登录页面默认用户名：admin 默认密码：password

登陆成功。

DVWA部署成功，可单击左侧下方的DVWA Security切换难易程度。
好啦，刷题愉快🤭！
